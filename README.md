# Copyright Evidence Organizer | 版权案件材料整理助手

这是一个面向互联网平台版权维权场景的法务实习 Demo。现在包含两个互补模块：

- Copyright Evidence Organizer：版权案件材料整理助手
- Rights Document Checklist：版权授权文件审核清单工具

它把“侵权线索整理、证据材料核对、权属文件归档、授权链审核、案件台账统计”做成一个打开即用的小工具。

## 使用方式

1. 打开 `index.html`。
2. 在顶部切换两个模块。
3. 点击“载入样例”查看演示数据，或录入自己的案件/文件。
4. 系统会自动生成：
   - 案件登记表
   - 证据材料清单
   - 版权文件归档清单
   - 案件进度统计表
   - 待补充材料提醒
   - 案件线索文件解析结果
   - 授权文件审核台账
   - 文件正文解析结果
   - 授权字段提取结果
   - 文件补正提醒
5. 可导出 CSV，或用“打印/保存为 PDF”生成汇报材料。

## 案件材料文件解析

在“版权案件材料整理”模块中，可以上传线索表、取证记录或 CSV 台账。系统会尝试从 `.docx`、`.txt`、`.csv` 中提取：

- 作品名称
- 权利人
- 侵权链接
- 平台
- 发现时间
- 截图情况
- 权属文件情况
- 处理进度

识别到的字段会自动回填左侧案件表单；无法稳定识别的字段会显示为“待人工确认”。PDF 暂时只记录文件名。

## Rights Document Checklist 功能

选择文件类型后，系统会自动给出对应核查项：

- 授权书：主体一致、授权期限、信息网络传播权、维权授权、转授权、签章、附件清单。
- 版权登记证书：作品名称、权利人、作品类别、登记时间、证书编号、作品发表证明。
- 合作协议：合同主体、合作期限、权利归属、维权/转授权条款、附件和盖章页。
- 营业执照：主体名称、统一社会信用代码、有效期、法定代表人、扫描件完整性。
- 身份证明：签署人身份、证件有效期、正反面/关键信息页、配套委托文件。
- 链路授权文件：授权链连续性、逐层主体一致、期限覆盖、权利范围、转授权、附件完整性。

正文解析支持范围：

- `.docx`：尝试读取 Word 正文并提取授权相关字段。
- `.txt`：读取纯文本并提取授权相关字段。
- `.csv`：读取表格文本并提取授权相关字段。
- `.pdf`：暂时只记录文件名，不读取正文。

系统会尽量提取权利人/授权人、被授权人、作品名称、授权期限、授权范围、信息网络传播权、维权/投诉/诉讼授权、转授权和附件清单。无法稳定识别的字段会标记为“待人工确认”，不会直接判断为缺失。

重要提示：本工具仅用于材料整理和审核线索提示，不构成法律意见；最终结论需由法务人员结合原件、业务背景和适用规则复核。

## 简历表述

独立搭建版权案件材料整理与授权文件审核 Demo，围绕互联网平台版权维权场景，设计案件登记、证据清单、权属文件归档、授权链核查、文件补正提醒和进度统计等模块，可辅助完成侵权线索整理、材料核对、版权文件审核和案件台账管理。

## 1 分钟面试演示话术

“我把岗位描述中的日常工作拆成两个模块。第一个模块处理侵权线索和案件材料，录入侵权链接、作品、权利人、截图和材料状态后，系统自动生成案件登记表、证据清单、归档清单和待补提醒。第二个模块处理版权授权文件审核，针对授权书、版权登记证、合作协议、营业执照、身份证明和链路授权文件，自动给出主体一致、授权期限、信息网络传播权、转授权、签章和附件完整性等核查项。这个 Demo 的重点不是复杂技术，而是把法务实习生容易接触到的材料审核逻辑流程化，减少漏项，方便交接和统计。”

演示顺序建议：

1. 点击案件模块的“载入样例”，展示案件登记、证据清单和待补材料提醒。
2. 切到授权文件审核模块，选择“授权书”或“链路授权文件”，展示 checklist 如何随文件类型变化。
3. 保存一条“需补正”的审核记录，展示系统如何自动生成补正提醒。
4. 导出 CSV 或打印为 PDF，说明结果可用于交接、汇报和归档。

## 对应岗位要求

- 协助准备案件证据和材料：自动生成证据材料清单与待补提醒。
- 检索调查取证：记录侵权链接、平台、发现时间、截图/录屏状态。
- 案件登记统计：生成案件登记表与进度统计表。
- 审核版权文件并归档：记录权属文件状态并给出归档核对建议。
- 审核授权链条和合同附件：按文件类型提示主体、期限、权利范围、转授权、签章和附件完整性核查项。
